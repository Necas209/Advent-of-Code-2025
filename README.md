# Advent-of-Code-2025

Source code for the [2025 edition of Advent of Code](https://adventofcode.com/2025).

The solutions are written in C# 14 (.NET 10). To run a solution for a specific day, navigate to the repository's root
and
run the following command (where `XX` is the day number):

```bash
cd src/DayXX && dotnet run && cd -
```
